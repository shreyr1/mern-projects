My Blog

Welcome to my simple blog made with HTML

Post 1: Learning HTML

Today I created my very first blog page using only HTML. HTML (HyperText Markup Language) is the backbone of every website. It is used to structure text, images, and links on a webpage.

While making this page, I learned how to use headings, paragraphs, images, and links. The most important lesson was understanding how HTML tags work together. This might look simple, but it is the foundation of web development.

Post 2: Why I Like Coding

Coding is not just about writing lines of code; it’s about solving problems and building something useful. Every time I write a program, I feel like I’m teaching the computer to do something new.

What excites me most about coding is the creativity it allows. With just a few commands, I can design a page, build a calculator, or even create a game. In the future, I want to learn CSS to make my websites look beautiful and JavaScript to add interactivity.

Post 3: My Future Goals

My goal is to become a full-stack web developer. This means I want to learn both the front-end (designing websites) and the back-end (working with databases and servers).

To achieve this, I plan to practice regularly, build small projects like portfolios and blogs, and later move on to larger applications. One day, I hope to create websites and apps that help people in their daily lives.

This blog is just the beginning of my coding journey, and I’m excited to see where it takes me!
